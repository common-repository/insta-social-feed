== Installation ==



Starting with Dew Social Feed consists of just two steps: installing and setting up the shortcode. Dew Social Feed is designed to work with your site's specific needs, so don't forget to go through the Dew Social Feed configuration wizard as explained in the 'after activation' step!

= INSTALL USER REGISTRATION FROM WITHIN WORDPRESS =
1. Visit the plugins page within your dashboard and select 'Add New';
2. Search for 'Dew Social Feed';
3. Activate 'Dew Social Feed' from your Plugins page;
4. Go to 'after activation' below.

= INSTALL USER REGISTRATION MANUALLY =
1. Upload the 'dew-social-feed' folder to the /wp-content/plugins/ directory;
2. Activate the 'Dew Social Feed' Plugin through the 'Plugins' menu in WordPress;
3. Go to 'after activation' below.

= AFTER ACTIVATION =
1. You should see the Dew Instagram Feed submenu in Setting menu;
2. Go through the Dew Social Feed and set up the plugin for your site;
3. You're done!
